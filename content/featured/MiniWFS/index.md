A persistent, Unix-like block-based filesystem built in C using inodes, allocation bitmaps, and direct plus indirect block addressing — interoperable with standard Unix tools on a raw disk image. Full VFS integration via FUSE implements file, directory, and metadata operations with a **100% automated test pass rate**, extended with POSIX-compliant timestamps and custom per-file extended attributes.
