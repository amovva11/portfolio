# Abhijay Movva — Personal Portfolio

My personal portfolio site, built with [Gatsby](https://www.gatsbyjs.org/) and
[styled-components](https://styled-components.com/).

The design is based on the open-source [v4](https://github.com/bchiang7/v4) template by
[Brittany Chiang](https://brittanychiang.com), used under the MIT License.

## 🚀 Getting Started

```sh
# install dependencies
yarn install

# start the dev server at http://localhost:8000
yarn develop

# build for production
yarn build
```

## 🛠 Where to edit my content

- **Hero / intro** — `src/components/sections/hero.js`
- **About + skills** — `src/components/sections/about.js`
- **Experience** — `content/jobs/*/index.md`
- **Featured projects** — `content/featured/*/index.md`
- **Other projects / archive** — `content/projects/*.md`
- **Contact** — `src/components/sections/contact.js`
- **Email & social links** — `src/config.js`
- **Site title / SEO** — `gatsby-config.js`
- **Resume** — drop a PDF at `static/resume.pdf`
- **Photo** — replace `src/images/me.jpg`
- **Project cover images** — replace `content/featured/*/demo.png`
